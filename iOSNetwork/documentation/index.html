benCoding.Network Module

Description

This module provides support for Carrier information, SSID, BSSID, and Apple’s Reachability utility class.

Before You Start

  • This is an iOS module designed to work with Titanium SDK 1.8.1 or greater.
  • Before using this module you first need to install the package. If you need instructions on how to install a 3rd party module please read this installation guide.
  • If you would like access to the source code, please visit us on github https://github.com/benbahrenburg/TiNetworkHelpers..

Accessing the benCoding.Network Module

To access this module from JavaScript, you would do the following:

var networkHelpers = require('bencoding.network');

Components

CarrierInfo

CarrierInfo provides access to the native CTTelephonyNetworkInfo framework object.

Ti.API.info("Create a Carrier Info Object");
var carrierInfo = networkHelpers.createCarrierInfo();

Learn more about this component here

CurrentNework

CurrentNetwork provides access to the native CNCopyCurrentNetworkInfo object

Ti.API.info("Create a Current Network Info Object");
var currentNetwork = networkHelpers.createCurrentNetwork();

Learn more about this component here

Reachability

The Reachability component provides a wrapper for the Apple Reachability module. For more about Reachability please see the Apple documentation here.
