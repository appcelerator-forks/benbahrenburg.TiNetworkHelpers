benCoding.Network Titanium Module

Using the Reachability Sub Module

Description

The Reachability component provides a wrapper for the Apple Reachability module. For more about Reachability please see the Apple documentation here.

For a detailed example please see the app.js included in the example folder.

Accessing the benCoding.Network Module

Before you can start using any of the module's functionality you need to use the require keyword to make the module accessible from JavaScript. To access this module from JavaScript, you would do the following:

var networkHelpers = require('bencoding.network');

 

Creating an instance of the Reachability Sub Module

Ti.API.info("Create a Reachability Object");
var reachability= networkHelpers.createReachability();

 

Object Methods

Method Name Description
reachabilityWithHostName This method provides a boolean result if Local Wifi is enabled.
reachabilityForLocalWiFi This method takes a host name, such as www.apple.com and returns a boolean result if the host is reachable.
hostNameReachableBy This method takes a host name, such as www.benCoding.com and returns a constant with how it is reachable.  This method returns a reachability constant shown below.

Object Constants

Constant Name Description
NETWORK_STATUS_NOT_REACHABLE Constant indicating that the host is not reachable. This constant is returned by the hostNameReachableBy method.
NETWORK_STATUS_WIFI Constant indicating that the host is reachable by WiFi. This constant is returned by the hostNameReachableBy method.
NEWORK_STATUS_WWAN Constant indicating that the host is reachable over the mobile network. This constant is returned by the hostNameReachableBy method.

Example

var networkHelpers = require('bencoding.network');

Ti.API.info("Create a Reachability Object");
var reachability= networkHelpers.createReachability();

Ti.API.info("Get reachabilityWithHostName => " + reachability.reachabilityWithHostName("www.apple.com"));

Ti.API.info("Get reachabilityForLocalWiFi => " + reachability.reachabilityForLocalWiFi());

Ti.API.info("Get check how we are able to reach apple.com ");
var hostReachableByType = reachability.hostNameReachableBy("www.apple.com");
Ti.API.info("Apple.com was reachable by => " + hostReachableByType );

//Create a JavaScript helper function

function getReachableType(findType){
var results = "Oops something went wrong";

if(findType==reachability.NETWORK_STATUS_NOT_REACHABLE){
results="Not Reachable";
}

if(findType==reachability.NETWORK_STATUS_WIFI){
results="WiFi Network";
}

if(findType==reachability.NEWORK_STATUS_WWAN){
results="Mobile Network";
}

return results;
};

var reachableType=getReachableType(hostReachableByType);

Ti.API.info("Reachable via => " + reachableType);
