benCoding.Network Module

Method: reachabilityForLocalWiFi

Description

The reachabilityForLocalWiFi method provides a boolean result if Local Wifi is enabled.

For a detailed example please see the app.js included in the example folder.

Accessing the benCoding.Network Module

Before you can start using any of the module's functionality you need to use the require keyword to make the module accessible from JavaScript. To access this module from JavaScript, you would do the following:

var reachability = require('bencoding.network').createReachability();

Using reachabilityForLocalWiFi

The below example shows how to create the Reachability object and check if the device has a reachable WiFi connection.

Ti.API.info("See if we have a current WiFi connection");
Ti.API.info("Get reachabilityForLocalWiFi => " + reachability.reachabilityForLocalWiFi());

 

Why do I use this method?

There are several reasons to check if your local WiFi connection is reachable.  The most common is to ensure you have an assumed faster connection speed.
