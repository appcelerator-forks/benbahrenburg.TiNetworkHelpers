benCoding.Network Module

Method: reachabilityWithHostName

Description

The reachabilityWithHostName method takes a host name, such as www.apple.com and returns a boolean result if the host is reachable..

For a detailed example please see the app.js included in the example folder.

Accessing the benCoding.Network Module

Before you can start using any of the module's functionality you need to use the require keyword to make the module accessible from JavaScript. To access this module from JavaScript, you would do the following:

var reachability = require('bencoding.network').createReachability();

Using reachabilityWithHostName

The below example shows how to create the Reachability object and check if the host name, "www.apple.com",  is reachable.

Ti.API.info("Get reachabilityWithHostName => " + reachability.reachabilityWithHostName("www.apple.com"));

 

Why do I use this method?

This method can be used to "ping" a host name to make sure it is reachable by the device before a connect is attempted.  

Arguments

Name Type Summary
hostName String See above for example

Return Value

  • Boolean true/false returned dependingon if the host name provided is reachable by the device.
